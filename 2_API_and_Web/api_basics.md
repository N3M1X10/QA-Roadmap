# Веб-технологии и API 🌐

## Клиент-серверная архитектура
Фундамент работы интернета. Взаимодействие всегда происходит по принципу: **Запрос (Request) -> Ответ (Response)**.

* **Клиент (Client):** Устройство или программа на стороне пользователя (браузер, мобильное приложение, десктопная программа). Отвечает за отображение интерфейса и отправку запросов.
* **Сервер (Server):** Удаленный мощный компьютер, который хранит данные, обрабатывает логику работы системы и возвращает клиенту нужный результат.

*Пример:* Приложение погоды на телефоне (Клиент) отправляет запрос на сервер метеостанции. Сервер находит данные для текущего города и отправляет температуру обратно на телефон (Ответ).
