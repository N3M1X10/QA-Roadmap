# bug_reports.md

## Первый баг-репорт 🐛

### Описание (Summary)
Кнопка **отправки сообщения** в чате неактивна при вводе текста.

### Шаги воспроизведения (Steps to Reproduce)
1. Открыть главное окно чата.
2. Кликнуть в поле ввода сообщения.
3. Ввести текст "Привет".
4. Обратить внимание на кнопку "Отправить".

### Фактический результат (Actual Result)
Кнопка "Отправить" остаётся серой и не нажимается. Текст отправить невозможно.

### Ожидаемый результат (Expected Result)
После ввода хотя бы одного символа кнопка "Отправить" становится активной (меняет цвет) и отправляет сообщение при клике.
